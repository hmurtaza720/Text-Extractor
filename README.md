# Text Extractor Web App

A powerful text extraction and analysis tool powered by OCR and Google Gemini AI.

## Features
- **Upload & Extract**: Support for Images (PNG/JPG) and PDF.
- **AI-Powered**: Uses Gemini 1.5 Flash for intelligent text correction and formatting.
- **Search & Tagging**: Organize documents with tags and find them instantly.
- **Mobile Friendly**: Fully responsive design.

## Deployment
Deployed on Hugging Face Spaces using Docker.
